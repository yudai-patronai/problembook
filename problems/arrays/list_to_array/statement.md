Имеется массив, реализующий односвязный список следующим образом.

Элементами списка являются кортежи `(value, next)`. А формат массива следующий:

    linked_list = [(value, next), (value, next), (value, next), ...]

Здесь `value` - значение элемента списка, `next` - индекс в массиве следующего элемента списка.
У последнего элемента списка индекс `next == -1`.

Вам необходимо написать функцию `list_to_array(linked_list, head)`, которая извлекает из списка значения его элементов и возвращает их в виде массива.

### Аргументы функции

Функция `list_to_array(linked_list, head)` имеет два аргумента

- `linked_list` - массив (`<class 'list'>`), хранящий список в указанном выше формате;
- `head` - кортеж (`<class 'tuple'>`), представляющий собой головной элемент списка `linked_list`.

### Возвращаемое значение

Функция `list_to_array(linked_list, head)` должна вернуть массив `<class 'list'>`.

### Примеры работы функции

    >>> l = [(40, 3), (21, 0), (17, 1), (77, -1), (17, 1)]
    >>> head = (17, 1)
    >>> list_to_array(l, head)
    [17, 21, 40, 77]
