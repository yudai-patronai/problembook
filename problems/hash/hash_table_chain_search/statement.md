Определить, есть ли элемент в хеш-таблице с цепочками.

### Формат входных данных

Два числа, которые нужно сложить.

### Формат выходных данных

Одно число — результат.

### Примеры

```
-> 1 2
--
<- 3
```

```
-> 1 -1
--
<- 0
```
