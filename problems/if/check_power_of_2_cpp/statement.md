Проверка числа на степень двойки

### Формат входных данных

Число n (0 < n < 10^9)

### Формат выходных данных

Строка YES в случае, если n является некоторой степенью двойки, NO в противоположном случае

### Примеры

```
-> 1024
--
<- YES
```

```
-> 5
--
<- NO
```
