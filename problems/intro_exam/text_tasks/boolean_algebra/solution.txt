Укажите значения переменных K, L, M, N, при которых логическое выражение

    (L ∨ M) ∧ (¬K → M) ∧ ¬N ∧ ¬M

истинно.

Здесь обозначены

- ¬ отрицание
- ∨ дизъюнкция
- ∧ конъюнкция
- → импликация

Решение. 
N = 0, M = 0 видно из конъюкции в конце. Из первой скобки тогда L = 1. Во второй скобке - K = 1, поскольку из не-К должна следовать ложь (M = 0), а ложь следует только из лжи.

Решение 2.
Упростить выражение…

Ответ. K = 1, L = 1, M = 0, N = 0.
