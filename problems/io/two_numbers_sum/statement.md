Сложить два числа.

### Формат входных данных

В строке последовательно вводятся два числа, которые нужно сложить.

### Формат выходных данных

Одно число — результат.

### Примеры

```
-> 1 2
--
<- 3
```

```
-> 1 -1
--
<- 0
```
