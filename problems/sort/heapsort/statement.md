Напишите функцию `heapsort(A)`, выполняющую сортировку кучей массива чисел `A`.

### Формат входных данных

Два числа, которые нужно сложить.

### Формат выходных данных

Одно число — результат.

### Примеры

```
-> 1 2
--
<- 3
```

```
-> 1 -1
--
<- 0
```
