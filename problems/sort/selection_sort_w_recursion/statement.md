В этой задаче необходимо реализовать сортировку выбором с использованием рекурсии.

### Формат входных данных

Два числа, которые нужно сложить.

### Формат выходных данных

Одно число — результат.

### Примеры

```
-> 1 2
--
<- 3
```

```
-> 1 -1
--
<- 0
```
