Найдите подстроку, заключенную в двойные кавычки.

Гарантируется, что такая подстрока в исходной есть и притом единственна.

### Формат входных данных

Одна строка - исходная.

### Формат выходных данных

Одна строка.

### Примеры

```
-> 1 2
--
<- 3
```

```
-> 1 -1
--
<- 0
```
