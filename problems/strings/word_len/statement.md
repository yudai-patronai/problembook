Сложить два числа.

### Формат входных данных

В строке, состоящей из слов, разделенных пробелами, определить длину кратчайшего и самого длинного слов. Длина слов не превышает 1000.

### Формат выходных данных

Одна строка

### Примеры

```
-> AAAA bbb aa
--
<- 2 4
```
