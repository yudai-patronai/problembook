Необходимо найти НОД двух чисел, используя алгоритм Евклида.

### Формат входных данных

На вход в одной строке подаются два числа.

### Формат выходных данных

Одно число - НОД входных чисел.

### Примеры

```
-> 30 18
--
<- 6
```

```
-> 1071 462
--
<- 21
```
