Необходимо найти НОД двух чисел, используя алгоритм Евклида.

### Формат входных данных

На вход подаются два натуральных числа, по числу в новой строке.

### Формат выходных данных

Одно число - НОД входных чисел.

### Примеры

```
-> 30
-> 18
--
<- 6
```

```
-> 1071
-> 462
--
<- 21
```
