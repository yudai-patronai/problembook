Необходимо написать функцию prime(n), генерирующую n-ое простое число, где n - натуральное. Первым простым числом является 2, вторым - 3, третьим - 5 и так далее.

### Формат входных данных

Натуральное число - n

### Формат выходных данных

n-ое простое число

### Примеры

```
-> 6
--
<- 13
```

```
-> 100
--
<- 173
```
