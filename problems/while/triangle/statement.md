Дается число N > 0, необходимо вывести треугольник высоты N из чисел.
Пример для N = 3:
1
2 3
4 5 6 

### Формат входных данных

Число 0 < N < 50

### Формат выходных данных

Полученный треугольник. Цифры в каждой строке разделены одним пробелом.

### Примеры

```
-> 2
--
<- 1
<- 2 3
```

```
-> 3
--
<- 1
<- 2 3
<- 4 5 6 
```
